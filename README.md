### Stak, Lib
1. Base on Kotlin + Coroutines + Flow
2. Compose -> View
3. Hilt -> dependency injection
4. Room -> Local API
5. OkHttp3 + Retrofit2 -> Server API 
6. Glide -> Image
7. Moshi -> Json Parsing
8. DataStore -> Local Data Storage
9. Paging3 -> Data List
